# 执行步骤

## 阶段 1：项目骨架

目标：建立最小可运行桌面应用。

事项：

- 初始化 Electron + React + TypeScript + Vite 项目。
- 配置主进程、preload、渲染进程。
- 打开一个 Windows 桌面窗口。
- 准备基础目录结构。

验收：

- 可以通过开发命令启动应用窗口。
- 窗口展示 ZZmusic 初始界面。

## 阶段 2：本地音乐库

目标：完成导入和持久化。

事项：

- 使用系统文件选择器导入单个或多个音频文件。
- 过滤支持的音乐格式。
- 避免重复导入相同路径。
- 将播放列表保存到本地 JSON。
- 应用启动时读取历史播放列表。
- 支持从播放列表移除歌曲。

验收：

- 关闭应用后再次打开，歌曲列表仍然存在。
- 移除歌曲只影响播放器列表，不删除磁盘文件。

## 阶段 3：播放核心

目标：完成常规播放器基础能力。

事项：

- 使用 `HTMLAudioElement` 播放本地文件。
- 实现播放、暂停、上一首、下一首。
- 实现进度条拖动。
- 显示当前时间和总时长。
- 实现音量控制。

验收：

- 可以连续播放导入的歌曲。
- 拖动进度条能正确跳转。
- 音量调整即时生效。

## 阶段 4：界面打磨

目标：达到第一版可用和风格统一。

事项：

- 完成侧边栏 + 主区域 + 底部控制条布局。
- 完成浅色 Apple Music 风格界面。
- 处理空状态、当前播放状态、不可用状态。
- 检查窗口缩放下的布局稳定性。

验收：

- 界面清爽统一。
- 常用操作入口清晰。
- 文本和控件没有明显重叠。

## 阶段 5：打包准备

目标：为 Windows x64 发布做准备。

事项：

- 添加 Windows x64 打包配置。
- 配置应用图标、应用名、版本号。
- 验证打包产物可打开。

验收：

- 生成可在 Windows x64 上运行的桌面应用产物。
