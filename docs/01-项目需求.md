# ZZmusic 项目需求

## 项目定位

ZZmusic 是一个只面向 Windows x64 的本地桌面音乐播放器。第一阶段目标是先做出稳定可运行的基础架构，后续再逐步注入更多功能。

## 第一版范围

必须实现：

- 像普通 Windows 软件一样打开运行。
- 支持导入单个或多个本地音乐文件。
- 支持播放列表持久化，关闭应用后再次打开仍能看到之前导入的歌曲。
- 支持播放、暂停、上一首、下一首。
- 支持拖动进度条跳转播放位置。
- 支持显示当前播放时间和总时长。
- 支持音量控制。
- 支持从播放列表移除歌曲，但不删除磁盘原文件。
- 使用类似 Apple Music 的布局：侧边栏 + 主区域 + 底部播放控制条。

## 第一版暂不做

- 文件夹批量扫描。
- 歌词。
- 专辑封面解析。
- 播放历史。
- 收藏。
- 全局快捷键。
- 系统托盘。
- 迷你播放器。
- 均衡器。
- 账号、云同步或联网功能。

## 默认音频格式

第一版优先支持：

- `mp3`
- `wav`
- `flac`
- `m4a`

实际可播放能力以 Electron/Chromium 在 Windows 上的音频支持为准。
