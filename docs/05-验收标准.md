# 验收标准

## 第一版完成标准

第一版只有在满足以下条件后才算完成：

- 应用可以在 Windows x64 开发环境中启动。
- 可以导入一个或多个本地音乐文件。
- 导入后的歌曲显示在播放列表中。
- 重启应用后播放列表仍存在。
- 可以播放、暂停、上一首、下一首。
- 可以拖动进度条跳转。
- 可以调整音量。
- 可以从播放列表移除歌曲，且不会删除原音乐文件。
- 当前播放歌曲有可见状态。
- 空列表时有清晰导入入口。
- 界面符合浅色、简洁、现代的设计方向。

## 每次开发后的检查

每次开发结束前至少检查：

- 是否能启动应用。
- 本次改动的核心功能是否能手动验证。
- 是否更新当天开发日志。
- 是否有新增待办需要写入日志或 docs。

## 不通过条件

出现以下任一情况，不视为稳定完成：

- 应用无法启动。
- 导入后重启丢失播放列表。
- 移除歌曲会删除磁盘原文件。
- 播放控制按钮无反馈。
- 进度条拖动后播放位置错误。
- 界面主要区域出现明显文字或控件重叠。
